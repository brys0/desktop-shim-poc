# Setup

1. Download libmpv from [here](https://sourceforge.net/projects/mpv-player-windows/files/libmpv/)
2. Extract libmpv-2.dll to desktop-shim-poc/
3. Rename libmpv-2.dll to libmpv.dll 

# Running

`go run ./window/main.go`
